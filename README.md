# Employee-Management-System
Our project Employee Management System is a desktop-based application developed to efficiently manage employee records within an organization. The system provides a user-friendly interface to perform essential employee operations such as adding, updating, deleting, and viewing employee details in a structured manner.
The application is built using Python (Tkinter) for the graphical user interface and MySQL as the backend database to ensure secure and reliable data storage. It helps reduce manual work, improves data accuracy, and enhances overall productivity in employee data management.

Key Features

Add new employee details

Update existing employee records

Delete employee information

View employee details in tabular format

User-friendly GUI using Tkinter

Secure data storage using MySQL database

Technologies Used

Programming Language: Python

GUI Framework: Tkinter

Database: MySQL

Libraries: PIL (for image handling), mysql-connector

Project Use Case

This system is suitable for small to medium organizations to manage employee data digitally, replacing traditional paper-based or spreadsheet-based methods.
